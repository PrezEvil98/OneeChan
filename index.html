How do I get started?

  1. Install 4chan X -- Mayhem's or ihavenoface's.
  2. Refresh the 4chan board and enable "Persistent QR" and "Autohide QR" in 4chan X's settings.
  3. Follow the instructions below for your browser, and report any issues you find.

Chrome

  1. Install Tampermonkey.
  2. Install OneeChan with Tampermonkey.

Firefox

  1. Install Greasemonkey (Scriptish is fine, but hasn't been updated since January).
  2. Restart Firefox.
  3. Install OneeChan.
